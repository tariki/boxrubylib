# Boxrubylib

Ruby library of OpenBox API.

## Installation

Add this line to your application's Gemfile:

    gem 'boxrubylib'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install boxrubylib

## Usage

Add this line to your application's code:

    require 'boxrubylib'
